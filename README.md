# 🍔 McDonald's Sales & Menu Data Analysis

> A complete end-to-end data analysis project exploring McDonald's menu items, order patterns, revenue trends, and category-wise performance — backed by an interactive Excel dashboard.

---

## 📊 Project Overview

This project analyzes **12,234 order records** from McDonald's alongside a structured menu dataset to uncover business insights such as:

- 💰 Total revenue and average order value
- 🏆 Best-selling categories and items
- 📅 Sales trends across days of the week and months
- 🕐 Peak ordering times throughout the day

---

## 📁 Dataset Description

| File | Description |
|------|-------------|
| `mcdonal_s_menu_items.xlsx` | Menu catalog with item ID, name, category, and price |
| `order_details.csv` | 12,234 order records with date, time, and item references |
| `dashboard.xlsx` | Interactive Excel dashboard with pivot-based visuals |

### Menu Categories
`Burger` · `Chicken` · `Pasta` · `Sandwich` · `Sides` · `Fries` · `Breakfast` · `Salad` · `Wraps` · `Shakes`
